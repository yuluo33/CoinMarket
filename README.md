# CoinMarket

CoinMarket 是一个基于 SwiftUI 开发的 macOS 加密货币行情监控应用，提供主窗口与状态栏双入口，适合在桌面侧边常驻查看核心币种价格与收藏列表。

## 项目定位

这个项目更偏向“轻量桌面盯盘工具”，而不是完整交易终端。

它的核心目标是：
- 用尽量简洁的界面展示常用币种行情
- 通过状态栏持续显示收藏币种价格
- 在不打断日常工作的前提下快速查看市场变化

## 核心特点

- **主窗口 + 状态栏双入口**
  - 可在主窗口中查看完整币种列表、搜索、收藏与设置
  - 可在 macOS 状态栏中直接查看收藏币种价格轮播
- **单主窗口恢复机制**
  - 从状态栏、Dock 或状态栏弹层重新打开时，都会复用同一个主窗口逻辑
  - 避免重复打开多个主界面导致内容异常或状态不一致
- **关闭窗口不退出应用**
  - 点击左上角关闭按钮后，主窗口会被隐藏
  - 应用仍保持运行状态，并继续显示在 Dock 与状态栏中
- **收藏驱动的状态栏展示**
  - 状态栏价格轮播仅针对已收藏币种
  - 没有收藏时，状态栏会提示“添加收藏”
- **本地持久化**
  - 收藏、语言、价格单位、刷新间隔等设置都会保存在本地
  - 应用重启后会自动恢复
- **多语言与多法币**
  - 同时支持多语言界面和多价格单位显示
  - 适合不同地区用户快速切换查看习惯

## 窗口与运行逻辑

这是当前版本比较重要的一部分：

### 主窗口行为

- 应用只维护一个主窗口逻辑入口
- 点击状态栏价格、状态栏弹层“打开主窗口”按钮、Dock 图标时，都会尝试唤起同一个主窗口
- 如果主窗口已存在且被最小化，会恢复它
- 如果主窗口被关闭隐藏，会重新显示它

### 关闭按钮行为

- 点击左上角关闭按钮后：
  - 主窗口会关闭显示
  - 应用不会退出
  - Dock 仍显示应用处于运行状态
  - 状态栏入口继续保留
- 之后可以通过以下方式再次打开主窗口：
  - 点击 Dock 图标
  - 点击状态栏价格文字
  - 点击状态栏弹层中的“打开主窗口”按钮

### 最小化按钮行为

- 最小化仍然保留 macOS 默认行为
- 最小化与关闭是两种不同动作，互不混淆

## 数据源与刷新机制

### 行情数据源

- **Binance API**
  - 用于获取币种 24 小时行情和最新价格
  - 当前主要使用 Binance 的 `ticker/24hr` 接口

### 汇率数据源

- **Frankfurter API**
  - 用于获取 USD 到其他法币的汇率
  - 当前会用它把行情价格换算成 CNY、KRW、JPY、VND 等单位

### 刷新策略

- 币种行情按用户设置间隔自动刷新
- 汇率数据大约每小时刷新一次
- 若已有旧数据，网络请求失败时界面会尽量保持已有内容，减少空白状态

## 默认行为与本地存储

应用会使用 `UserDefaults` 持久化以下内容：
- 收藏币种集合
- 自动刷新间隔
- 状态栏轮播间隔
- 价格单位
- 语言

当前默认值：
- 语言：中文
- 价格单位：USDT
- 自动刷新间隔：5 秒
- 状态栏轮播间隔：5 秒

## 技术实现特点

- **开发框架**：SwiftUI
- **架构模式**：MVVM
- **状态管理**：Combine + `@StateObject` / `@ObservedObject`
- **异步方式**：Swift Concurrency (`async/await`)
- **系统能力**：AppKit + SwiftUI 混合，用于管理状态栏、弹层与窗口行为
- **本地化方案**：自定义本地化管理器
- **本地持久化**：`UserDefaults`

## 使用说明

### 主窗口使用

1. 启动应用
2. 等待行情加载完成
3. 在顶部搜索框输入币种名称或符号
4. 点击星标管理收藏
5. 点击右上角齿轮进入设置

### 状态栏使用

1. 启动后查看状态栏文字
2. 若尚未收藏币种，会显示“添加收藏”
3. 添加收藏后，状态栏会按轮播间隔显示不同收藏币种价格
4. 点击状态栏弹层中的价格单位菜单，可以快速切换显示单位
5. 点击“打开主窗口”可恢复主界面

## 注意事项

### 1. 当前不是完整市场列表

当前版本并不是从全市场动态排序，而是基于预设币种列表展示 20 个主流币种。

### 2. 图表组件已存在，但当前主流程未完整启用

项目中已经包含价格图表视图组件，但当前 Binance 数据提供逻辑没有返回 7 天 sparkline 数据，因此图表能力暂未成为现版本主打功能。

### 3. 汇率换算依赖第三方接口

如果 Frankfurter 接口不可用，非 USD / USDT 单位的价格换算可能不够及时或会回退到默认汇率表现。

### 4. 网络环境会直接影响加载结果

如果 Binance 或 Frankfurter 在当前网络环境中受限，应用可能出现：
- 加载失败
- 数据延迟
- 汇率未及时刷新

### 5. 关闭窗口不等于退出应用

这是当前版本的设计行为：
- 关闭主窗口后，应用继续驻留后台
- 如果需要完全退出，请使用 Dock 菜单退出、应用菜单退出，或快捷键退出

## 当前版本总结

当前版本最适合以下使用场景：
- 需要在 macOS 桌面长期观察少量核心币种价格
- 习惯通过状态栏快速查看收藏币种波动
- 希望在关闭主窗口后仍保留后台常驻能力
- 希望使用多语言、多价格单位进行轻量查看
